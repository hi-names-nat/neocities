Making Games

I've been making games for about a decade now. I started in 2017 because one of my friends mentioned they eventually wanted to make a game company. We began to talk and agreed there was no reason to wait, so we grabbed another mutual friend and began to work on our first project (fun fact, it's the same project Magic on the Music Page is from.)

That project and group fizzled, but I found that I really enjoyed making games, so when I found myself in University I quickly glommed on to the university game developers group, even running it for a few years near the end, and educating others on how to use game engines.

Recently, I've been working on starting a new game developers group, and am interested in making a new small company for new development.

And hey, if you like the stuff here and wanna know how something works, or have a game development question, you can go ahead and @ Me on Twitter or Bluesky, or send me an ask on Tumblr and we can chat! (Public chats only if you're a minor, please.)
