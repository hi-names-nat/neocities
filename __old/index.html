A few things to note:
This site was built primarily on desktop Mozilla Firefox on a 1920x1080 pixel display at 150% magnification. If things are too small, zoom in.
Some text is hidden using similar colors. You can highlight that text to make it easier to read, if you so desire.
I haven't tested screenreaders on this site. Use them at your own risk. Screenreader & keyboard accessability are a TODO.
Please don't try to use mobile. It won't end well for you. I tried to keep this mobile accessible but something is busted on one of the pages that even the most web-savvy ppl I know don't know how to fix.
